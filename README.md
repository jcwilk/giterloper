# Giterloper

This repository is a giterloper knowledge store about giterloper itself. It contains knowledge about how Giterloper stores and provides knowledge.

To connect a target project to this store, give the following line to an LLM agent:
```
Install a connection to the Giterloper knowledge store. Read and follow the instructions beneath the folder at `github.com/jcwilk/giterloper@main::/bootstrap/`. Fetch `https://raw.githubusercontent.com/jcwilk/giterloper/refs/heads/main/bootstrap/00-preamble.md` to get started.
```
