# Background

Giterloper is a Git-backed knowledge storage system. The name is a play on "interloper": the system is universal and can be used by any agent or tool; it interlopes into projects via connection to a knowledge store.

The main repository is both the source of the paradigm (constitution, bootstrap, tooling) and a knowledge store about giterloper itself. Knowledge is stored in a Git repository so it benefits from versioning, branching, and merging without custom infrastructure.
